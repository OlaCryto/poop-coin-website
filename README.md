$POOP Coin - Official WebsiteWelcome to the official GitHub repository for the $POOP coin website. This project is a fully functional, multi-page Flask web application designed to serve as the online home for the ultimate meme coin on the AVAX blockchain.💩 About the ProjectThis website provides a complete online presence for the $POOP meme coin. It's designed to be fun, engaging, and informative, giving potential investors and community members everything they need to know about the project. The site is built with a mobile-first, responsive design to ensure it looks great on all devices.Live Site(You can add your live Render URL here once it's deployed!)✨ Key FeaturesModern & Responsive Design: A professional, mobile-friendly layout that adapts to any screen size.Interactive Frontend: Features dynamic animations, sound effects, and a raining poop emoji background to create an engaging user experience.Web3 Integration: Includes wallet connection functionality using ethers.js to interact with the Avalanche C-Chain.Community Meme Gallery: Users can upload their own memes, which are displayed in a gallery sorted by community votes.Whitelist Functionality: An API endpoint allows users to submit their wallet addresses to a whitelist for early access.Multi-Page Structure: Includes dedicated pages for About, Tokenomics, How to Buy, Launch Info, and Community links.🛠️ Tech StackThis project was built using the following technologies:Backend: Python with the Flask web framework.Frontend: Standard HTML, CSS, and JavaScript.Styling: Tailwind CSS for a utility-first design approach.Web3: Ethers.js for wallet connectivity.Data Storage: Simple JSON files for managing meme and whitelist data.Deployment: Hosted on Render with a Gunicorn production server.🚀 Setup and Local InstallationTo run this project on your local machine, follow these steps:Clone the repository:git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
cd YOUR_REPOSITORY_NAME
Create a virtual environment (Recommended):python -m venv venv
